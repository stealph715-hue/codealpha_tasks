# CodeAlpha Full Stack Development Internship - Tasks

This repo contains my completed tasks for the CodeAlpha Full Stack
Development internship.

## Tasks completed

### Task 1 - Simple E-commerce Store
`/Task1_Ecommerce_Store`
A basic e-commerce site with product listings, shopping cart, checkout,
order processing, and user registration/login. Built with Django + HTML/CSS/JS.

### Task 2 - Social Media Platform
`/Task2_Social_Media_Platform`
A mini social media app with user profiles, posts, comments, likes, and a
follow system. Built with Django + HTML/CSS/JS.

### Task 3 - Project Management Tool
`/Task3_Project_Management_Tool`
A Trello/Asana-style collaborative tool with projects, task boards, task
assignment, comments, and in-app notifications. Built with Django + HTML/CSS/JS.

## How to run any task
Each task folder is a standalone Django project with its own README, e.g.:
```
cd Task1_Ecommerce_Store
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```

## Tech stack
- Backend: Django (Python)
- Frontend: HTML, CSS, JavaScript
- Database: SQLite
